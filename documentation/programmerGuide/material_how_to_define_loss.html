Material how to define loss

For material definition there are params:

  • Epsilon - relative electric permittivity (must be >=1)
  • Mue - relative magnetic permeability (must be >=1)
  • Kappa - electric conductivity (must be >=0)
  • Sigma - magnetic conductivity (non-physical property, must be >=0)

Here is formula how to calculate Kappa from loss parameter which is normally in microwave RF called "tan(Delta)": 

                    Kappa = atan(2*pi*f*tan(Delta))
                    pi = 3.14159265...
                    f = frequency at which tan(Delta) is required to have value as it have
                    tan(Delta) = material loss parameter which is provided by manufacturer
As seen, tan(Delta) in openEMS for user defined material could be simulated on one frequency, for frequency dependent loss parameter or frequency dependent permitivity or permeability must be used lorentz or debeye material.
